## Setup in Ubuntu

~~~ bash
# Install pip 
~ $ sudo apt-get install python-pip

# Verify pip 
~ $ pip -V
pip 8.1.1 from /usr/lib/python2.7/dist-packages (python 2.7)
~~~

## Setup in CentOS

~~~ bash
# Install pip
~ $ yum -y install python-pip

# Verify pip
~ $ pip -V
~~~

## Setup in MacOS

~~~ bash
# Install pip
~ $ sudo easy_install pip
~~~
